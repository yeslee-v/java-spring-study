# 테스트 케이스
## GET 유저 별 도서 목록 나열
1. 유저가 로그인한 상태인지 검증한다
2. 전체 상태 별 갯수를 반환하면서, 각 상태로 필터링한 책 목록을 반환한다: 기본 상태값은 IN_PROGRESS, 응답 상태 sucess
3. 상태 별 각 도서 목록에는 책 이름, 저자, 출판사, 읽은 페이지, 전체 페이지, 진행률을 반환한다.
4. 반환할 책 목록이 없다면 빈 배열을 반환한다
5. ~~반환할 책 목록이 10개가 넘어가면 페이지네이션으로 10개씩 반환한다~~
6. 각 도서 목록은 생성 시간 역순(createdAt desc)으로 정렬한다
	1. 수정 시간 역순으로 하면 재정렬하는데 cost가 꽤 들거 같아 pass
