- Collections
- [ArrayList](https://www.w3schools.com/java/java_arraylist.asp): 흔히 아는 배열(array)와 다르다
```java
import java.util.ArrayList;

public class Main {
	public static void main(String[] args) {
		ArrayList<String> fruits = new ArrayList<String>();

		// to add elements to the list
		fruits.add("strawberry");
		fruits.add("banana");
		
		// add an item at a specific position using index number
		fruits.add(1, "apple");
		
		// to access an element in the ArrayList
		fruits.get(0); // strawberry

		// to modify an element
		fruits.set(1, "grape");

		// get ArrayList size
		fruits.size();

		// get each element, use for-each loop
		for (String fruit: fruits) {
			System.out.println("fruit name: " + fruit);
		}

		// to remove an element
		fruits.remove(0);
				
		// to remove all the elements
		fruits.clear();
	}
}
```
