# 1768. Merge Strings Alternately
## 6ms 나옴 → 시간을 단축시켜보자!
```java
class Solution {
	public String mergeAlternately(String word1, String word2) {
		int minLen = 0, maxLen = 0;
		String longerWord = "";
	
		if (word1.length() < word2.length()) {
			minLen = word1.length();
			maxLen = word2.length();
			longerWord = word2;
		
		} else {
			minLen = word2.length();
			maxLen = word1.length();
			longerWord = word1;
		}
		
		String answer = "";
		int i = 0;
	
		for (i = 0; i < minLen; i++) {
			answer += word1.charAt(i);
			answer += word2.charAt(i);
		}
		
		if (i < maxLen) {
			answer += longerWord.substring(i);
		}
		
		return answer;
	}
}
```

### `StringBuilder` 사용: 0ms 나옴..!
- `String` 매번 더하기 = 매번 새 `String` 객체 생성
```java
import java.lang.StringBuilder;

class Solution {
	public String mergeAlternately(String word1, String word2) {
		int minLen = 0, maxLen = 0;
		String longerWord = "";
		
		if (word1.length() < word2.length()) {
			minLen = word1.length();
			maxLen = word2.length();
			longerWord = word2;
		} else {
			minLen = word2.length();
			maxLen = word1.length();
			longerWord = word1;
		}

		StringBuilder sb = new StringBuilder();
		int i = 0;
				
		for (i = 0; i < minLen; i++) {
			sb.append(word1.charAt(i));
			sb.append(word2.charAt(i));
		}

		if (i < maxLen) {
			sb.append(longerWord.substring(i));
		}
	
		return sb.toString();
	}
}
```
### 코드를 좀 더 간소화해보자: 변수 선언 최소화
```java
import java.lang.StringBuilder;

class Solution {
	public String mergeAlternately(String word1, String word2) {
	StringBuilder sb = new StringBuilder();
	
	int word1Len = word1.length();
	int word2Len = word2.length();
	int maxLen = Math.max(word1Len, word2Len);
	
	for (int i = 0; i < maxLen; i++) {
		if (i < word1Len) {
			sb.append(word1.charAt(i));
		}
		
		if (i < word2Len) {
			sb.append(word2.charAt(i));
		}
	}

	return sb.toString();
	}
}
```

# 1071. Greatest Common Divisor of Strings
- str1, str2를 나눌 수 있는 공통된 부분을 찾는거라고 생각하고 풀음
```java
import java.lang.StringBuilder;

class Solution {
	public String gcdOfStrings(String str1, String str2) {
		StringBuilder sb = new StringBuilder();
		int str1Len = str1.length();
		int str2Len = str2.length();
		int minLen = str1Len < str2Len ? str1Len : str2Len;
		
		for (int i = 0; i < minLen; i++) {
			if (str1.charAt(i) == str2.charAt(i)) {
				if (sb.toString().indexOf(str1.charAt(i)) == -1) {
					sb.append(str1.charAt(i));
				}
			}
		}
		
		return sb.toString();
	}
}
```
